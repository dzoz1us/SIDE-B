# Руководство пользователя
## Автоматизированная информационная система «SIDE B — Музыкальный магазин»

### 1. Начало работы
Откройте браузер и перейдите по адресу `http://localhost:5173` (среда разработки). На главной странице отображаются популярные ансамбли и лидеры продаж. Через верхнее меню доступны разделы «Ансамбли», «Пластинки», «Лидеры продаж», «Филиалы» и кнопка «Войти».

### 2. Регистрация и вход
Для регистрации нажмите «Войти» → «Зарегистрироваться». Заполните поля «Имя», «Фамилия», «Email», «Пароль» и «Подтверждение пароля». Нажмите «Зарегистрироваться». После регистрации вы будете авторизованы автоматически. Для входа введите email и пароль на странице входа.

### 3. Просмотр каталога
В разделе «Ансамбли» можно искать коллективы по названию, фильтровать по типу и стране. При клике на ансамбль открывается страница с вкладками: «О коллективе», «Дискография», «Состав». В разделе «Пластинки» доступен поиск по названию и ансамблю, фильтрация по лейблу, наличию и филиалу. Клик на пластинку открывает подробную информацию и треклист.

### 4. Бронирование пластинок
1. Авторизуйтесь в системе
2. Перейдите в раздел «Пластинки» и выберите пластинку
3. Нажмите «Забронировать»
4. В модальном окне выберите филиал для получения
5. Нажмите «Подтвердить бронь»

Бронь действует 48 часов. Просмотреть брони можно в личном кабинете → «Мои брони». Для отмены нажмите «Отменить бронь» напротив активной брони.

### 5. Лидеры продаж и филиалы
Раздел «Лидеры продаж» показывает топ-10 пластинок текущего года. Раздел «Филиалы» содержит список магазинов с адресами, телефонами и часами работы.

### 6. Личный кабинет
После входа в меню пользователя (аватар в хедере) доступны:
- «Мои брони» — список всех броней с фильтрацией по статусу
- Смена пароля
- Редактирование профиля

### 7. Функциональное назначение команд меню

Верхнее меню приложения содержит следующие команды:

| Команда           | Назначение                                   | Доступность |
|-------------------|----------------------------------------------|---------------------|
| SIDE B (лого)     | Переход на главную страницу                  | Всем                |
| Ансамбли          | Каталог ансамблей с поиском и фильтрацией    | Всем                |
| Пластинки         | Каталог пластинок с фильтрацией              | Всем                |
| Лидеры продаж     | Топ-10 пластинок текущего года               | Всем                | 
| Филиалы           | Список филиалов магазина                     | Всем                |
| Войти             | Страница входа в систему                     | Гостям              |
| Панель управления | Дашборд менеджера с быстрыми действиями      | Менеджерам, админам |
| Администрирование | Управление справочниками и учётными записями | Админам             |

Меню пользователя (аватар в правой части хедера) содержит команды:

| Команда           | Назначение                    | Доступность         |
|-------------------|-------------------------------|---------------------|
| Мои брони         | Просмотр и отмена броней      | Всем авторизованным |
| Панель управления | Быстрый доступ к дашборду     | Менеджерам, админам |
| Администрирование | Быстрый доступ к админ-панели | Админам             |
| Выйти             | Завершение сессии             | Всем авторизованным |

### 8. Структура и функциональное назначение пользовательских интерфейсов

**Главная страница.** Содержит приветственный баннер с названием магазина и кратким описанием, кнопки «Смотреть каталог» и «Ансамбли». Ниже расположена секция «Популярные ансамбли» — сетка карточек коллективов с фотографией, названием и жанром. Далее — секция «Лидеры продаж» с топ-3 пластинок текущего года. В нижней части — баннер с призывом зарегистрироваться для бронирования.

*(Скриншот: главная страница)*

**Каталог ансамблей.** В верхней части — строка поиска и фильтры по типу коллектива, стране и сортировке. Результаты отображаются в виде карточек с фотографией, названием, типом, страной и статистикой (количество альбомов и участников). Поддерживается пагинация.

*(Скриншот: каталог ансамблей)*

**Страница ансамбля.** Верхняя часть — баннер с фотографией и основной информацией (название, страна, год основания, количество альбомов и участников). Три вкладки: «О коллективе» (описание), «Дискография» (сетка обложек пластинок с ценами), «Состав» (список музыкантов с ролями и инструментами).

*(Скриншот: страница ансамбля, вкладка «Дискография»)*

**Каталог пластинок.** Строка поиска и фильтры по лейблу, наличию, филиалу и сортировке. Пластинки отображаются в виде сетки карточек с обложкой, названием, ансамблем, годом выпуска, ценой и бейджем наличия. Клик на карточку открывает страницу пластинки.

*(Скриншот: каталог пластинок)*

**Страница пластинки.** Слева — крупная обложка, справа — название, ансамбль, год выпуска, лейбл, каталожный номер, цена, наличие. Ниже — треклист в виде таблицы с номерами дорожек, названиями произведений, композиторами и длительностью. Кнопка «Забронировать» (для авторизованных) или «Войдите, чтобы забронировать» (для гостей).

*(Скриншот: страница пластинки)*

**Личный кабинет.** Меню слева: «Профиль», «Мои брони», «Смена пароля». Страница «Мои брони» содержит фильтр по статусу и список карточек броней с информацией о пластинке, филиале, датах и статусе. Активные брони можно отменить.

*(Скриншот: страница «Мои брони»)*

**Панель менеджера.** Дашборд со статистикой (всего пластинок, активных броней, продано за год) и кнопками быстрых действий. Разделы: «Управление пластинками» (таблица с возможностью добавления, редактирования, удаления), «Активные брони» (таблица с кнопками «Выдать» и «Снять»), «Склад» (остатки по филиалам), «Офлайн-продажа» (поиск пластинки и фиксация продажи), «Статистика» (графики и таблицы).

*(Скриншот: панель менеджера)*

**Административная панель.** Сетка карточек-разделов: «Ансамбли», «Музыканты», «Произведения», «Филиалы», «Менеджеры», «Логи действий». Каждый раздел содержит таблицу с данными и кнопками управления (добавить, редактировать, удалить). Раздел «Менеджеры» дополнительно позволяет блокировать и разблокировать учётные записи.

*(Скриншот: административная панель)*
