# Техническое задание
## Автоматизированная информационная система «SIDE B — Музыкальный магазин»

### 1. Основание для разработки
Задание на курсовое проектирование по дисциплине «Создание программного обеспечения».

### 2. Цель разработки
Создание веб-приложения для автоматизации учёта товаров, бронирования и продаж в музыкальном магазине, специализирующемся на торговле виниловыми пластинками и компакт-дисками.

### 3. Функциональные требования
- Хранение информации об ансамблях, музыкантах, произведениях и их взаимосвязях
- Ведение каталога пластинок с учётом оптовых и розничных цен, остатков на складе и статистики продаж
- Регистрация и аутентификация пользователей
- Бронирование пластинок авторизованными пользователями с выбором филиала и автоматическим резервированием на 48 часов
- Управление товарами менеджерами: добавление, редактирование, удаление пластинок, фиксация офлайн-продаж, обработка броней, просмотр склада и статистики
- Управление справочной информацией администратором: ансамбли, музыканты, филиалы, учётные записи менеджеров, просмотр логов
- Отображение лидеров продаж текущего года

### 4. Нефункциональные требования
- Клиент-серверная архитектура с REST API
- Серверная часть на Python/Django, клиентская — на React/TypeScript
- База данных под управлением PostgreSQL
- Аутентификация по JWT-токенам
- Разграничение прав доступа по четырём ролям: гость, авторизованный пользователь, менеджер, администратор
- Время отклика API не более 2 секунд

### 5. Роли пользователей

| Роль                        | Права                                                                                  |
|-----------------------------|----------------------------------------------------------------------------------------|
| Гость                       | Просмотр каталога, поиск, фильтрация, регистрация, вход                                |
| Авторизованный пользователь | Бронирование пластинок, просмотр и отмена броней, личный кабинет                       |
| Менеджер                    | Управление пластинками, офлайн-продажи, обработка броней, просмотр склада и статистики |
| Администратор               | Управление ансамблями, музыкантами, филиалами, менеджерами, просмотр логов             |
